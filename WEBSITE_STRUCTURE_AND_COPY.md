## Design System / Styling Notes

**Color Palette**:
- Primary: Slate (dark gray)
- Secondary: Moss (green)
- Accent: Sand (beige/tan)
- Background: Bone/Cream (light beige)
- Text: Slate with various opacities

**Typography**:
- Headings: Serif font (editorial style)
- Body: Sans-serif font (light weight)
- Small text: Uppercase with letter-spacing
- Sizes: Very small (9-10px) to very large (responsive viewport-based)

**Spacing**:
- Large section padding: py-32 to py-40
- Max width containers: 1800px
- Grid gaps: 16-24px typically

**Animations**:
- Framer Motion used throughout
- Fade in, slide up animations
- Hover effects on interactive elements
- Scroll-based parallax effects

**Images**:
- All images use Next.js Image component
- Aspect ratios: 4:5 for portraits, 3:4 for testimonials
- Object positioning: Various (50% 20%, 50% 25%, etc.)
- Overlays: Subtle dark overlays for texture

---

## Key Features

1. **Smooth Scrolling**: React Lenis for smooth scroll behavior
2. **Responsive Design**: Mobile-first, breaks at lg (1024px)
3. **Form Validation**: Zod schema validation on contact form
4. **Email Integration**: Form submissions via Resend API
5. **SEO Optimized**: Hidden text for search engines, proper alt tags
6. **Performance**: Next.js Image optimization, lazy loading

---

## Important Notes for Implementation

- All section IDs match navigation links
- "Services" and "Experience" both link to #experience (same section)
- Form requires all fields and validates before submission
- Images should be optimized and placed in `/public/images/` directory
- All text uses specific typography classes for consistent styling
- Hover states and transitions are important for the premium feel
- The site uses a very editorial, high-end aesthetic with lots of white space
